    grid_database¶

    数据库管理模块

    提供统一的数据库操作接口: - 持仓数据持久化 - 网格交易会话管理 - 交易记录管理

    初始化数据库管理器

    参数：

    名称 类型 描述 默认
    db_path str

    数据库文件路径,默认使用config.DB_PATH

    		 None
